# Proyecto Final - UTN

Aplicación web desarrollada como entrega formal para la Universidad Tecnológica Nacional (UTN). Consiste en una interfaz moderna construida con React y Vite.

---

## Tecnologías utilizadas

- **React** — Biblioteca de JavaScript para interfaces de usuario
- **Vite** — Entorno de desarrollo ultrarrápido
- **Node.js** — Entorno de ejecución (versión recomendada: 18 o superior)

---

## Instrucciones de ejecución

Asegurate de tener instalado [Node.js](https://nodejs.org/) antes de comenzar.

### 1. Clonar el repositorio *(opcional)*

Si descargaste el proyecto como archivo ZIP, podés saltear este paso. Si no, clonalo con:

```bash
git clone URL_DE_ESTE_REPOSITORIO
```

### 2. Instalar las dependencias

Ejecutá el siguiente comando en la raíz del proyecto (donde está el archivo `package.json`):

```bash
npm install
```

### 3. Iniciar el servidor de desarrollo

```bash
npm run dev
```

Luego abrí tu navegador en la dirección que indique la terminal (habitualmente `http://localhost:5173`).
